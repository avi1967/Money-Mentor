# Impact Model (Assumption-Based)

## Baseline Problem

- Most users lack structured financial planning.
- Human advisor model (₹25,000+/year) is inaccessible for mass market.

## Assumptions

- 10,000 monthly active users in ET ecosystem pilot.
- 35% complete at least one full planning workflow per month.
- Manual planning effort without tool: 2.5 hours/user/month.
- With AI mentor: 20 minutes/user/month.
- Implied user time value: ₹300/hour.

## Time Saved

- Saved per active planner = 2.5h - 0.33h = 2.17h/month.
- Planners/month = 10,000 x 35% = 3,500.
- Total hours saved = 3,500 x 2.17 = 7,595 hours/month.

## Monetary Value of Time

- 7,595 x ₹300 = **₹22,78,500/month** user productivity value.

## Advisor Cost Avoidance Proxy

- If 15% of planners would otherwise pay entry-level advisor fee:
- Users = 3,500 x 15% = 525
- Annual cost avoided = 525 x ₹25,000 = **₹1,31,25,000/year**

## Revenue Recovery/Upside (Optional)

- If 8% users adopt recommended tax actions saving avg ₹18,000/year:
- 10,000 x 8% x ₹18,000 = **₹14,40,00,000/year** tax leakage reduction.

## Conclusion

Even conservative assumptions show strong value across:

- Time efficiency
- Affordability and inclusion
- Tax optimization and better long-term investing behavior
